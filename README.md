# Mova

An interpreted language designed around ownership and safe memory management, yet easy to use.

## Examples

Here is an example to get you started:

```
let x = 123
let y = 111

fn add_two_numbers(x, y) = x + y // adds four

add_two_numbers(x, y) // returns 234
```
